# Recep Özgür Mıh - Portfolio

An AI-driven mobile engineer portfolio featuring advanced web interactions, a 3D background system, and multi-language support.

## 🌐 Live Access

The portfolio is accessible via:
*   **Custom Domain:** [recepozgur.com](https://recepozgur.com)
*   **GitHub Pages:** [recepzgrmh.github.io](https://recepzgrmh.github.io)

## 🚀 Key Features

*   **3D Particle Background:** A high-performance network system built with **Three.js** that reacts to scroll, mouse movement, and theme changes.
*   **Dynamic i18n:** Built-in zero-dependency translation system supporting English and Turkish.
*   **Spotify Live Integration:** Real-time display of currently playing tracks via a custom Cloudflare Worker.
*   **Easter Egg:** A secret physics-based "destruction" mode. Hold click on the particle sphere for 8 seconds to witness the UI collapse under simulated gravity.
*   **Modern UI/UX:** Glassmorphism aesthetics, GSAP-powered animations, and a custom cursor system.
*   **Optimized Performance:** 100% WebP image assets and efficient spatial grid calculations for particle connections.

## 🛠️ Tech Stack

*   **Core:** Vanilla JavaScript (ES6+), HTML5, CSS3
*   **3D Engine:** [Three.js](https://threejs.org/)
*   **Animations:** [GSAP (GreenSock)](https://greensock.com/gsap/)
*   **Styling:** Custom CSS + Tailwind CSS utilities
*   **Infrastructure:** GitHub Pages, Cloudflare Workers (for Spotify API)

## 📁 Structure

*   `/tr/`: Turkish version of the site.
*   `/en/`: English version of the site.
*   `/apps/`: Showcase of production-ready mobile applications.
*   `/blog/`: Technical deep-dives and engineering articles.
*   `/lib/`: Core libraries and integration scripts.
*   `/components/`: Reusable UI components like the Spotify Live Card.
